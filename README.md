# MeetMode 🚀

MeetMode is an AI-powered networking platform built for the **Build Beyond Limits 2.0 Hackathon**.

It helps people discover and connect with nearby attendees who are open to meaningful networking at hackathons, conferences, meetups, universities, and professional events.

# Problem Statement

Networking at events is often difficult because attendees don't know:

- Who is open to talking
- Who shares similar interests
- How to start meaningful conversations

MeetMode solves this through AI-powered recommendations and consent-first networking.

# Project Flow

1. User signs up
2. User logs in
3. User creates a profile
4. User joins an event
5. User enables Meet Mode
6. AI recommends matching attendees
7. User chats and builds connections
