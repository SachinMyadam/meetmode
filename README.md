# 🚀 MeetMode - AI Powered Networking Platform

Live App: https://meetmode.vercel.app  
Backend API: https://meetmode-backend.onrender.com  

## 📌 Features
- User Signup/Login (JWT Authentication)
- Protected Routes
- AI Networking Concept
- Events System
- Chat Ready Architecture
- Notifications Support

## 🏗️ Tech Stack
Frontend: React (Vite), Vercel  
Backend: Node.js + Express, Render  
Auth: JWT  
Database: Custom + Redis (Valkey ready)

## 🔐 Auth Flow
1. Signup
2. Login
3. JWT stored in localStorage
4. Protected routes
5. /auth/me verification

## 🌐 API
- /auth/signup
- /auth/login
- /auth/me
- /auth/logout
