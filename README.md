# MeetMode

MeetMode is a simple social networking and event platform built for the Build Beyond Limits 2.0 hackathon.

## What it does

- Login, signup, logout
- Dashboard with stats and event cards
- Friends search and match cards
- Event register flow
- Chat page
- QR networking
- Notifications
- AI assistant popup on dashboard
- Profile and settings pages

## Architecture

Frontend (React + Vite)
-> API client
-> Backend (Express)
-> Valkey / Redis compatible storage

## Tech Stack

Frontend: React, Vite, React Router DOM, CSS, React Icons, QRCode React  
Backend: Node.js, Express, JWT, bcryptjs, dotenv, ioredis  
Database: Valkey  
Tools: Git, GitHub, Linux terminal, npm
